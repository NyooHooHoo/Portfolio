Arrays

An array is a data structure that stores multiple values of the same data type under one name in memory. These individual values, or “elements,” are accessed with the name of the array and the index of a specific element in square brackets. Arrays are always zero-indexed, so the first element will have index 0, second index 1, and so on (e.g. arr[0] of ['a', 'b', 'c'] is 'a'). The main benefit of arrays reduce the number of individual variables needed to store large amounts of data. They are declared and initialized using the following syntax:

anyType [] arrayName = new anyType[length]; OR {value1, value2, etc.};

Elements initialized using the first method have the default values 0 for numbers, false for booleans, and null for objects such as String. The length of an array is a public attribute that can be accessed directly by arrayName.length. Parallel arrays are several arrays of same length that work together to store related data that may be of different types, and are a further application of the efficient nature of arrays.

A two-dimensional (2D) array is an array of arrays that can organize data into a grid of rows and columns instead of only a linear, or one-dimensional (1D), format. The declaration and initialization of 2D arrays is very similar to 1D ones.

anyType [][] arrayName = new anyType[rows][columns]; OR {{value1, value2, etc.}, {value3, value4, etc.}, ...};

While 1D arrays can easily be traversed (loop through every value) using a single loop, 2D arrays must be travered using a nested loop, a loop within another loop.

public DogRatings2D(Scanner d) {
    dogBreeds = new String[10];
    ratings = new int[5][10];
    categories = new String[]{"maintenance", "energy", "lifespan", "amicability", "appearance"};
    for (int i = 0; i < dogBreeds.length; i++) {
        dogBreeds[i] = d.nextLine();
        for (int j = 0; j < dogBreeds[i].length; j++) {
            ratings[j][i] = d.nextInt();
        /* Rest of loop not shown */
        }
    }
}

Constructor from another class in Ratings assignment, demonstrating 2D arrays with the variable ratings initialized with 5 rows for categories, which used to be individual 1D arrays, and 10 columns for scores. The 2D array is then traversed with a nested for-loop to initialize its elements with values read from a file. The check i < dogBreeds.length represents limit of rows and j < dogBreeds[i].length in the inner loop represents the limit within each row.

Common Mistakes

  • Accessing the length of an array as a method like Strings arrayName.length() instead of as an attribute arrayName.length.
  • Forgetting zero-indexing and using 1 for the first element instead of 0
  • Causing an ArrayIndexOutOfBoundsException by using arrayName.length as the last index instead of arrayName.length - 1 or using a i <= arrayName.length in a loop instead of i < arrayName.length.
