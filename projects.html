Projects

Code Depot

A website where users can find online courses to support their learning in various topic in coding. Features include a homepage displaying courses with a search bar, a favourites page to save courses, and a like/dislike system to rate courses. Users create accounts and log in to have their data stored in the back-end built with the Python Flask framework and SQL. The frontend was build with HTML/CSS and JavaScript. LyonHacks II Hackathon - 3rd Place

Contributors: Aidan Wang, Luka Jovanovic, Sailesh Badri, Brian Song

Devpost Github

Spodissey

A web application that generates song reccomendations from countries all around the world for users based on their Spotify listening data. Users log in to their Spotify account and an interactive map UI is generated with highlighted countries that can be clicked to reveal a sidebar of recommended songs. Built in HTML/CSS, Javascript, and Typescript using Spotify's Developer API along with Leaflet API for maps. LyonHacks III Hackathon - 2nd Place

Contributors: Aidan Wang, Joshua Wang, Nirvan Rabbani, Patrick Bian

Devpost Github

ChemSim

An website for high school chemistry students to supplement their studying journey. Resources available include a molecule builder that generates IUPAC nomenclature and animated lessons teaching the main types of chemical reactions. Built with HTML/CSS and JavaScript using HTML Canvas and Cactus Web API to generate chemical names from SMILES generated strings.

Contributors: Aidan Wang, Sailesh Badri

Website Github

Creation Story

A Java program that tells an story of the universe, where a Creator populates the planet with Elemental spirits representing earth, air, water, and fire. OOP and inheritance is used to create four classes that extend from the Elemental class: Qarri, Hyrmus, Swalik, and Iklaff. Driver class initializes objects and outputs story to the console.

Contributors: Aidan Wang, Pouya Karimi, Sailesh Badri

MaCSBook

A Java program to store names, numbers, and grades of students. Methods to manipulate this data include calculating and displaying student averages, searching through students to change data, and sorting students alphabetically or by average mark using insertion and selection sorting algorithms.

Contributors: Aidan Wang, Lauren Hewang, Rita Xu, Matthew Tsui

Fantasy Game

A Java program simulating a single-player game with an elf that collects gold from castles to deliver to their dragon master or spend to buy healing potions. OOP and inheritance were used in the program with the abstract Character class being the parent of Elf and Dragon, each of which implement their own getName() method from the abstract class. Bridge

A Java program simulating the card game Bridge, using ArrayLists of string to represent a deck of cards and the hand of four players. The cards and randomly dealed in each iteration of the program and player scores are calculated using official Bridge rules to determine the winning player.

Contributors: Aidan Wang

Dog Ratings

A Java program storing data on the rating of ten dog breeds on a scale of 1-10 in the categories of maintenance, energy, lifespan, amicability, and appearance. These data are stored both using parallel arrays in one version of the program, and a single 2D array in another version. Methods available to the user include calculating the best dog breed overall and the best breed in each individual category.

Contributors: Aidan Wang, Lauren Hewang
