Inheritance

Inheritance is a feature in OOP that can be used to maximise code reusability between classes in a program. In a class relationship with inheritance, the Parent Class (or Superclass) is an class with general attrubites and methods that can be used in several subclasses. In Java, the Object class is the parent to all other classes written in the language, with very general methods such as toString(). The Child Class (or Subclass) is an extension of a parent class that inherits all the parent's public variables and methods, and can also override parent methods by redefining them. Child classes are defined using the syntax ChildName extends ParentName in the first line of the class. Child classes may only extend one parent class, but a class may have multiple children.

In the constructor(s) of child classes, super() is used to call a constructor of the parents class, as all objects of the child are also objects of the parent. Arguments in super() must match a pre-written constructor in the parent class. Finally, the this keyword is used to differentiate variables and methods with the same names in constructors.
