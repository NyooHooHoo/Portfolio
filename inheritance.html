Inheritance

  • Parent Class (or Superclass): abstraction, a class with general variables and methods that can be used in several subclasses
    • All Java classes extend from the Object class
  • Child Class (or Subclass): extension of a parent class, inherits all public variables and methods, can add additional and override existing
    • Is-A relationship between a child class and the parent class it extends, children may only extend one parent class, but a class may have multiple children
    • Has-A relationship between class and another class it implements
  • Constructor: super() must be called in child class to create parent class object
    • Arguments in super() must match a written constructor for parent
  • this keyword is used to differentiate variables and methods with the same names
